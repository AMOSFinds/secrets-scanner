How it Works

  1. Connect GitHub (OAuth) or use a Personal Access Token (read-only).
  2. Paste any repository URL and select a branch.
  3. We enumerate files via the GitHub API and scan in memory.
  4. Findings appear instantly. Optional Slack/email alerts on new leaks.
